## 📸 About the Project
A fun and functional browser-based Rock Paper Scissors game where you play against the computer. The game keeps track of your wins, losses, and ties — and your score is saved even after you close or refresh the browser using localStorage.

## ✨ Features
- 🪨 🗒️ ✂️ Click-based gameplay — choose Rock, Paper, or Scissors
- 🤖 Computer makes a random move every round
- 🏆 Real-time score tracking — Wins, Losses, and Ties
- 💾 **localStorage** — scores persist after page refresh or browser close
- 🔄 Reset button to clear scores and start fresh
- 📱 Mobile-friendly with responsive viewport

## 🛠️ Technologies Used
| Technology | Purpose |
|---|---|
| HTML5 | Game structure |
| CSS3 | Styling and layout |
| JavaScript (ES6) | Game logic, DOM manipulation, localStorage |

## 🧠 How It Works
1. Player clicks on Rock, Paper, or Scissors
2. Computer randomly selects a move using `Math.random()`
3. Winner is determined using conditional logic
4. Score updates on screen and is saved to `localStorage`
5. Scores remain saved even after closing the browser

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/karshu454545-dev/Rock-Paper-Scissor-Game.git
   ```
2. Open `index.html` in your browser.
